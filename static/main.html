接口测试页面导航

选择一个协议页面，快速验证非流式与 SSE 流式响应。

  • OpenAI 聊天模拟测试 使用 `/v1/chat/completions`，验证 OpenAI 兼容的请求体、分块内容和 `[DONE]` 结束标记。
  • 自定义流式响应测试 使用 `/v1/custom/chat`，验证自定义 `prompt_id`、回复字段和 `[FINISH]` 结束标记。
  • Anthropic Messages 模拟测试 使用 `/v1/messages`，验证 Anthropic 规范请求体、SSE 事件和 `message_stop` 结束事件。
